# notification_numbers

Custom-виджет Grist для регистрации извещений об изменении документации на изделия.

## Поля извещения

| Поле | Тип в Grist | Описание |
|---|---|---|
| Number | Text | Номер извещения (генерируется автоматически) |
| IssueDate | Date | Дата выпуска документа |
| Product | Ref (на таблицу изделий) | Изделие, для которого выпущено извещение |
| DocumentDesignation | Text | Обозначение документа, который был изменён |
| DocumentChangeNumber | Text | Номер изменения конкретного документа |
| ChangeReason | Text | Причина изменения (комментарий) |

Названия колонок сопоставляются через стандартный экран выбора колонок Grist —
физические имена колонок в вашей таблице могут отличаться.

## Настройка

1. Добавьте виджет в Grist через **Custom URL**, указав адрес `index.html`
   (например, через GitHub Pages этого репозитория).
2. В экране сопоставления колонок укажите таблицу и колонки для каждого поля.
3. Откройте настройки виджета (значок ⚙ в шапке формы) и укажите:
   - таблицу изделий, из которой выбирается «Изделие»;
   - колонку этой таблицы для отображения названия изделия;
   - префикс и разрядность номера извещения, начальный номер.

## Разработка

Виджет — один файл `index.html` без сборки (vanilla JS + Grist Plugin API).
Открывайте его напрямую как Custom Widget URL в Grist (например, через
`npx serve` локально или GitHub Pages в проде).
